2 Ошибка округления

Ограничения для Go:

  • 3 000 мс
  • 256 мб

Условие задачи

Маркетплейс получает комиссию p%p\%p% с каждой продажи товаров. Если был продан товар, который стоил aia_iai​ рублей, комиссия должна составить ai⋅p100a_i \cdot \frac{p}{100}ai​⋅100p​ рублей. По техническому заданию комиссия должна округляться в меньшую сторону до второго знака после запятой, то есть до целого числа копеек. Но, из-за допущенной программистом ошибки, комиссия всегда округлялась в меньшую сторону до целого числа рублей, то есть копейки отбрасывались. Найдите сумму, упущенную из-за ошибки, после продажи маркетплейсом nnn товаров со стоимостями aia_iai​ рублей.
Набор тестов

Входные данные
Стандартный ввод

Каждый тест состоит из нескольких наборов входных данных. Первая строка содержит целое число ttt (1≤t≤1051 \le t \le 10^51≤t≤105) — количество наборов входных данных. Далее следует описание наборов входных данных. Первая строка каждого набора входных данных содержит два целых числа nnn и ppp (1≤n≤1051 \le n \le 10^51≤n≤105, 1≤p<1001 \le p < 1001≤p<100) — количество проданных товаров и процент комиссии маркетплейса. Следующие nnn строк каждого набора содержат по одному целому числу aia_iai​ (1≤ai≤1091 \le a_i \le 10^91≤ai​≤109) — стоимости проданных товаров в рублях. Гарантируется, что сумма nnn по всем наборам входных данных не превосходит 10510^5105.

Выходные данные
Стандартный вывод

Для каждого набора входных данных на отдельной строке выведите ответ на задачу — сумму в рублях, упущенную маркетплейсом из-за ошибки. Это число должно содержать ровно\textbf{ровно}ровно два знака после точки (количество копеек). Комментарий к первому примеру: В первом наборе входных данных маркетплейс должен получить 111, 222, 333, 444 и 555 копеек комиссии, но из-за ошибки за каждый товар получено 0 копеек комиссии. Во втором наборе входных данных маркетплейс должен получить 40⋅5100=240 \cdot \frac{5}{100} = 240⋅1005​=2 рубля комиссии, копейки из-за ошибки не потеряны. В третьем наборе входных данных маркетплейс должен получить: ∙\bullet∙ За первый товар 50⋅99100=49.5050 \cdot \frac{99}{100} = 49.5050⋅10099​=49.50 рублей комиссии, из-за ошибки упущено 505050 копеек. ∙\bullet∙ За второй товар 1⋅99100=0.991 \cdot \frac{99}{100} = 0.991⋅10099​=0.99 рублей комиссии, из-за ошибки упущено 999999 копеек. Итого упущено 50+9950 + 9950+99 копеек, то есть 1.491.491.49 рублей.
